📌 Description

Dragon Tiger is a fast-paced online casino card game developed with Unity for Android and iOS platforms.

The game features realtime multiplayer gameplay, responsive betting systems, smooth card animations, and optimized mobile performance.

Players place bets on Dragon, Tiger, or Tie before cards are revealed. The side with the higher card wins the round.
